# Informes Semanales — Inventario Inteligente

En esta carpeta se guardan los informes semanales generados por el equipo. Cada informe se nombra como `informe-YYYY-WW.md` y contiene un resumen ejecutivo, commits relevantes, estado de integración hardware y tareas pendientes.

Usa el template `template-weekly.md` para crear nuevos informes.
